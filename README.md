# JAR Tracking Demo

This is a simple Vite + React + Tailwind project demonstrating a redesign concept for JAR Tracking. It includes:

- A hero section with calls to action
- Feature highlights
- A dashboard preview section
- Data & scale metrics
- Solutions overview

## Setup

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the browser at the address printed in the terminal.
